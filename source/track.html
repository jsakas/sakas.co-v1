Reflections Track Post

Posted April 20th, 2015

Today I launched a cool new feature on my site. It's an integration with Soundcloud that pulls tracks out of the API and puts them in blog posts. If you click play, there is a nice visualiser and start and stop button at the top of the page. From the user perspective it seems pretty simple, there are controls to play and stop, and a signal that shows them audio is playing (the visualiser). Behind the scenes, however, there are many different pieces of technology working together to. Some of the things that were necessary in order to create this feature were:

  • Extend my custom Mezzanine CMS theme to allow new types of blog posts called track posts, which allow custom fields to be entered by the user
  • Create a backend Python integration with the Soundcloud API to retrieve track information and the streaming URL
  • Create a front end JavaScript application that uses the another library I'm working on, AudioJS, to play and visualise the music using the awesome Web Audio API
  • Use pushState + Ajax (aka Pjax) to progressively enhance links on the page, which is how you can navigate the site and not interupt the music

In the future I want to extent this functionality to create a queue syste, as well as more full featured controls like seek, previous and next. These are currently all being worked on in the AudioJS project, but have not been implemented on this site as of yet. 

For my first music related post, I want to share a track I remixed a few years ago by Rudimental, called "Not Giving In". It's a great track and I had fun putting a little bit of a housey feel on it. You can download it for free from Soundcloud here: Rudimental - Not Giving In "Jontron Remix".

Enjoy!

-Jon
